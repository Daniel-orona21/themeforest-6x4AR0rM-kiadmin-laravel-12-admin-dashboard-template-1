Installation

Follow the steps below to set up your site template:

  1. Unzip the downloaded package and open the /ki-admin-laravel folder to find all the template files. You will need to upload these files to your hosting web server using FTP or localhost in order to use it on your website.
  2. Below is the folder structure and needs to be uploaded to your website or localhost root directory
    • ki-admin-laravel/resources/view - Contains all of the pages referenced
      • ki-admin-laravel/public/assets/css - Stylesheet files
      • ki-admin-laravel/public/assets/images - Images files
      • ki-admin-laravel/public/assets/js - Javascript files
      • ki-admin-laravel/public/assets/sass - Sass files
      • ki-admin-laravel/public/assets/vendor – All external libs.
    • ki-admin-laravel/resources/view/index.blade.php - Homepage
  3. first of all go to "https://getcomposer.org/download/" and then download composer
  4. Run command in project root
    • composer install
  5. Second install Node Modules:
    • npm install
  6. For start project use
    • php artisan serve
  7. You should upload all or specific .blade.php files as per your need.
  8. You are good to go for adding your content now!
  9. If you want to compile scss through vite,
    • - npm run dev
    • - npm run build

Laravel Structure

ki-admin follows a simple and easy to customize coding structure.

master.blade.php

                                            
<!DOCTYPE html>
<html lang="en">


    <!-- All meta and title start-->

               <div class="app-content">

                    <!-- header -->
                    @include('layout.header')

                    <main>

SCSS

We have added SASS .scss files in template. If you know how to use SASS you can change sass files and compile the css as well. You can find sass file here - Ki-admin/assets/sass/app

If you want to add new scss file. so you can_stylesheet.scss file in app folder and import file in style.scss and use easily

Open the sass/app/_variables.scss and Edit the values according to your needs. If you need more Advanced Setup then you can Edit the Respective Files yourself which have been branched inside the same Folder. It is completely at your discretion only to include the Required .scss Files you need to minimize the amount of CSS & including only the Styles of the Blocks you need. This can be setup in your style.scss File.
